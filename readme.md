
# Sonarqube: Automate Code Quality Analysis

### how to run 
* copy client/sonar-project.properties to project root
* adjust based on your requirements
* cd to project directory then run
* <SONAR_LOCATION>/client/sonar-scanner-4.0.0.1744-linux/bin/sonar-scanner
* view localhost:9000

### Error Solution
* Add more memory
* sudo sysctl -w vm.max_map_count=262144
